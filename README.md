# @eugeniuszx/react-native-media-picker

Cross-platform media picker for React Native (New Architecture) — no permission gallery access

## Installation


```sh
npm install @eugeniuszx/react-native-media-picker
```


## Usage


```js
import { multiply } from '@eugeniuszx/react-native-media-picker';

// ...

const result = multiply(3, 7);
```
